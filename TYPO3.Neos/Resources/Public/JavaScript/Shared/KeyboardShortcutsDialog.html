Keyboard shortcuts
General
  • Toggle fullscreen mode
    {{ctrlCmd}} + h

    esc
  • Toggle Edit/Preview panel
    {{ctrlCmd}} + e

Content Editing
  • Create new
    {{ctrlCmd}} + shift + a
  • Copy selected
    {{ctrlCmd}} + shift + c
  • Cut selected
    {{ctrlCmd}} + shift + x
  • Paste copied
    {{ctrlCmd}} + shift + v
  • Remove selected
    {{ctrlCmd}} + shift + d
  • Select next element
    tab
